## 2. 交互流程

### 选中文件
- 单击 → 选中高亮，取消其他
- Ctrl/Cmd + 单击 → 切换选中（多选）
- Shift + 单击 → 范围选中
- Ctrl/Cmd + A → 全选
- 点击空白 → 清空选中
- 双击 → 打开预览 / 进入文件夹

### 右键上下文菜单
文件菜单：打开 | 重命名(F2) | 下载 | 插入到聊天 | --- | 删除(Del)
空白菜单：新建文件夹 | 上传文件
多选菜单：删除选中项

### 键盘快捷键
| 快捷键 | 行为 |
|--------|------|
| ↑ / ↓ | 移动焦点 |
| Enter | 打开 |
| F2 | 重命名 |
| Delete | 删除 |
| Ctrl+A | 全选 |
| Escape | 清空选中 |

### 拖拽移动
- 拖文件 → 幽灵+数量badge → 拖到文件夹高亮 → 松手移动
- 从桌面拖入 → 上传（不变）

### 列排序
- 点击列头切换升/降序，显示排序箭头
- 文件夹始终在文件前面

### 批量操作
- 选中≥1时显示工具栏：已选中 N 个 | [删除] [取消]

### 图片缩略图
- 网格视图图片文件显示 CDN 缩略图
- 加载失败降级为 emoji

## 3. 状态设计

| 状态 | 显示 |
|------|------|
| 默认 | 正常 |
| 悬停 | `bg-[var(--s-hover)]` |
| 选中 | `bg-accent/10 border-l-2 border-accent` |
| 焦点 | `ring-2 ring-accent/30` |
| 拖拽中 | `opacity-50` |
| 放入目标 | `bg-accent/15 border-dashed border-accent` |
